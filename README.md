# hypaTorch
<p align="center">
<b>Highly Abstract Compound Networks in PyTorch</b>
</p>

This python package provides a flexible and comprehensive framework for creating compound PyTorch models in an abstract manner.

## Installation

```bash
uv add hypatorch
```
